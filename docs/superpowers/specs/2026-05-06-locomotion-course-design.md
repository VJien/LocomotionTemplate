# Locomotion 公开课设计文档

## 概述

面向游戏开发者（程序/策划/美术）的Locomotion系统公开课，时长1-1.5小时。以"从零搭建"为主线，逐步从单个动画循环构建到完整的角色运动系统。

## 核心信息

- **时长**: 1-1.5小时（课程内容约80-90分钟，QA 10-20分钟）
- **语言**: 中文撰写，技术术语保留英文原文
- **受众**: 游戏程序员、策划、美术（跨职能）
- **演示引擎**: Unreal Engine 5（基于Lyra Starter Game）
- **动画资源**: Paragon角色资产 + Lyra自带素材
- **交付物**: PPT课程文档 + UE5交互Demo项目

## 教学结构：三条并行线索

| 线索 | 载体 | 作用 |
|------|------|------|
| **理论线** | PPT | 原理、概念、设计思路 |
| **对比线** | 上线游戏素材 | 每个阶段展示真实游戏的locomotion表现，建立直觉 |
| **实践线** | UE5交互Demo | 可开关的完整搭建过程，听众亲手体验每层技术的差异 |

## UE5交互Demo设计

### 基础架构

- **基座**: Lyra Starter Game
- **角色**: Paragon角色
- **动画**: Paragon动画资产 + Lyra动画素材

### 交互UI面板

提供统一的开关面板，听众可逐步开启每个技术层：

| 开关 | 对应阶段 | 默认状态 |
|------|---------|---------|
| 4方向 + Orientation Warping | B | OFF（默认8方向Blend Space） |
| Distance Matching (Start) | C | OFF |
| Distance Matching (Stop) | D | OFF |
| Aim Offset | E | OFF |
| Turn In Place | F | OFF |
| Pivot | G | OFF |
| Jump | H | OFF |
| Lean | I | OFF |
| Foot IK | J | OFF |

所有开关OFF时 = 阶段B开头的"单个Loop"状态。逐步打开即可体验从零搭建的全过程。

### 测试场景

- 平坦地面（基础测试）
- 斜坡/台阶（IK测试）
- 开阔区域（转向/Pivot测试）
- 跳跃平台（Jump测试）
