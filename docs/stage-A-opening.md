## A2 — 什么是Locomotion

**形式**: 文字 + 图

**要点**:

- Locomotion = 角色在游戏世界中的移动与动画表现系统
- 不只是"走路"，而是整套运动系统：行走、跑步、启动、停止、转向、变向、地形适配……
- 好的Locomotion让玩家"感受不到它的存在"，差的Locomotion一眼就能看出来
- 它是程序、美术、策划三方协作的交汇点

**配图建议**: 左右对比图 — 左边角色僵硬滑步，右边角色自然流畅行走

## A4 — 课程路线图

**形式**: 横向时间线图表

**内容**:

```
Loop → Start → Stop → Turn In Place → Pivot → Foot IK → 完整系统
```

每一步标注：
| 步骤 | 解决的问题 | 引入的关键技术 |
|------|-----------|---------------|
| Loop | 让角色走起来 | Blend Space / OW |
| Start | 自然启动 | Distance Matching |
| Stop | 无滑步停止 | DM截断 / 左右脚选择 |
| Turn In Place | 原地自然转向 | TIP动画 |
| Pivot | 运动中变向 | Pivot动画 |
| Foot IK | 地形适配 | Foot IK |

**核心信息**: "我们会从最基础的单循环行走开始，一步步加入技术，最终搭建一个完整的Locomotion系统。每个阶段你都能看到问题是什么，以及我们怎么解决它。"
