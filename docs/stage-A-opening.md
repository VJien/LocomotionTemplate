## A2 — 什么是 Locomotion

**正文**:

```
Locomotion = 角色在游戏世界中的移动与动画表现

不只是"走路"
行走 · 跑步 · 启动 · 停止 · 转向 · 变向 · 地形适配

好的 Locomotion → 玩家感受不到它的存在
差的 Locomotion → 一眼就能看出来

程序 · 美术 · 策划 三方协作的交汇点
```

**配图**: 单个GIF动图 — 选一个流畅行走的角色循环，直观展示"这就是locomotion"

**备注**:
> 三层递进：先给一句话定义，然后用"好的vs差的"建立判断标准，最后点出跨职能协作。配图只需要一个自然流畅的行走动图，让观众看到"我们要讨论的就是这个"。不要用反面素材——反面素材留到各阶段讲问题时再展示，冲击力更强。

## A3 — 游戏 Locomotion 展示

**正文**:

```
（视频播放页，仅小标题）

这就是 Locomotion
```

**视频内容**: 3-5个游戏的locomotion片段，风格差异大
- 第三人称动作游戏 ×1（如TLOU）
- FPS ×1（如COD）
- 开放世界 ×1（如GTA V）
- 其他补充

**备注**:
> 不做评判，只展示"不同游戏都在解决这个问题"。选片原则：风格差异大，不同品类、不同时代。每个片段5-10秒，总时长控制在40秒内。让听众建立"这就是locomotion"的直觉。视频播放不方便可用GIF代替。

## A4 — 今天聊什么

**正文**:

```
（中心发散脑图，无多余文字）

中心：Locomotion

一级分支（向外发散）：
  Loop · Start · Stop · Aim Offset · Turn In Place
  Pivot · Jump · Lean · Foot IK · Motion Matching
```

**配图**: 中心"Locomotion"向外发散的脑图/气泡图，分支无序排列，不暗示先后顺序，展示话题广度

**备注**:
> 不需要告诉听众具体路线，让他们看到"我们需要讨论这么多东西"就够了。脑图风格偏轻松，视觉上让人感觉话题丰富但不压抑。每个分支可以配一个小icon或简笔小人暗示内容。后面每进入新阶段时可以高亮对应分支，起到进度锚点的作用。
